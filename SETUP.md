# Ad Superpowers — Setup Guide

## Quick Start (2 minutes)

### Step 1: Create your account
1. Go to [app.adsuperpowers.ai](https://app.adsuperpowers.ai)
2. Sign up with Google (one click)
3. You're on the **Free tier** — no credit card needed

### Step 2: Connect your ad platforms
1. In the dashboard, click **Connect Platform**
2. Select the platforms you want to connect (Meta Ads, Google Ads, GA4, Google Search Console, LinkedIn Ads, TikTok Ads, Google Tag Manager)
3. Complete the OAuth flow for each platform (authorize access)

### Step 3: Authentication
The MCP server uses OAuth — when you first make a request, your browser will open for authentication. No API key setup needed.

**Alternative: API Key authentication**
1. Go to [app.adsuperpowers.ai/settings](https://app.adsuperpowers.ai/settings)
2. Generate an API key
3. The plugin will prompt you to enter this key on first use

## Troubleshooting

### "Connection failed" or "Unauthorized"
- Make sure you've connected the relevant platform in the [dashboard](https://app.adsuperpowers.ai)
- Try re-authenticating by disconnecting and reconnecting the platform

### "No data returned"
- Verify the ad account has active campaigns with recent data
- Check that you're using the correct account ID
