# Workflow Optimizer

## Overview

Deze skill helpt bij het opzetten van gestroomlijnde workflows voor Meta Ads management, inclusief dagelijkse routines, wekelijkse reviews, maandelijkse audits en efficiënte processen voor campagne setup en optimalisatie.

## Dagelijkse Workflow

### Morning Check Routine (15 min)

```
DAGELIJKSE CHECKLIST - 09:00
============================

□ SPEND CHECK (2 min)
├── Is spend on pace? (niet te snel/traag)
├── Budget alerts checken
└── Anomalieën in spend pattern?

□ PERFORMANCE SNAPSHOT (5 min)
├── CPA vs target (groen/geel/rood)
├── ROAS vs target
├── Top 3 & bottom 3 ad sets
└── Conversie volume vs gisteren

□ DELIVERY CHECK (3 min)
├── Zijn alle campagnes actief?
├── Ad sets in learning phase
├── Rejected ads?
└── Budget caps bereikt?

□ QUICK ACTIONS (5 min)
├── Pauzeer duidelijke verliezers
├── Note maken voor later onderzoek
└── Urgente issues flaggen
```

### Quick Optimization Actions

```
SNELLE DAGELIJKSE ACTIES
========================

IF CPA > Target + 30%:
├── Check: Is dit 1 dag of trend (3+ dagen)?
├── Check: Voldoende conversies voor conclusie?
└── Action: Pause of budget reduceren

IF CPA < Target - 20%:
├── Check: Is audience niet uitgeput?
├── Check: Kan er geschaald worden?
└── Action: Budget +20% overwegen

IF Ad Set in Learning:
├── Check: Geen wijzigingen maken
├── Check: Op koers voor 50 conv/week?
└── Action: Monitoren, niet aanpassen

IF Rejected Ad:
├── Check: Welke policy violation?
├── Action: Fix en opnieuw indienen
└── Escalate: Als onterecht, appeal
```

## Wekelijkse Workflow

### Weekly Review Process (2-3 uur)

```
WEKELIJKSE REVIEW - [VASTE DAG]
===============================

BLOK 1: DATA ANALYSE (45 min)
─────────────────────────────
□ Export performance data (afgelopen 7 dagen)
□ Vergelijk met vorige week
□ Vergelijk met targets
□ Identificeer trends

ANALYSE VRAGEN:
├── Welke campagnes overperformen? Waarom?
├── Welke campagnes underperformen? Waarom?
├── Zijn er audience exhaustion signalen?
├── Hoe presteert nieuwe creative?
└── Welke placements leveren beste ROAS?

BLOK 2: CREATIVE REVIEW (30 min)
────────────────────────────────
□ Review ad performance metrics
□ Identificeer creative fatigue (CTR trend)
□ Plan nieuwe creative needs
□ Brief maken voor nieuwe assets

CREATIVE CHECKLIST:
├── Welke ads hebben >3 weken gedraaid?
├── Welke CTR's zijn dalend?
├── Welke hooks werken het best?
├── Welke formats presteren (video/static/carousel)?
└── Hoeveel actieve ads per ad set?

BLOK 3: REPORTING (30 min)
──────────────────────────
□ Update performance dashboard
□ Schrijf weekly summary
□ Highlight wins & learnings
□ Deel met stakeholders

BLOK 4: PLANNING (45 min)
─────────────────────────
□ Plan tests voor volgende week
□ Budget reallocatie indien nodig
□ Creative production pipeline update
□ Calendar check voor events/promos
```

## Maandelijkse Workflow

### Monthly Audit Checklist

```
MAANDELIJKSE ACCOUNT AUDIT
==========================

□ ACCOUNT HEALTH (30 min)
├── Pixel/CAPI status check
├── Event tracking validatie
├── Catalogus sync status
├── Audience sizes trend
└── Billing/payment status

□ CAMPAIGN STRUCTURE (30 min)
├── Zijn er te veel/weinig campagnes?
├── Ad set consolidatie nodig?
├── Naming conventions consistent?
├── Oude campagnes cleanup
└── Test campagnes evalueren

□ AUDIENCE AUDIT (30 min)
├── Custom audiences up-to-date?
├── Lookalike refresh nodig?
├── Audience overlap check
├── Exclusions correct?
└── Nieuwe audience ideeën

□ CREATIVE AUDIT (30 min)
├── Creative library opruimen
├── Performance per creative type
├── Top hooks documenteren
├── Creative refresh schedule
└── A/B test resultaten archiveren

□ BUDGET & BIDDING (20 min)
├── Budget efficiency per campagne
├── Bid strategy effectiveness
├── Dayparting opportunities?
├── Seasonality planning
└── Volgende maand budget planning

□ COMPETITIVE REVIEW (20 min)
├── Ad Library competitor check
├── Nieuwe messaging/offers?
├── Creative trends in markt
└── Opportunities identificeren
```

## Campaign Setup Workflow

### New Campaign Launch Checklist

```
CAMPAGNE LAUNCH CHECKLIST
=========================

PRE-LAUNCH (1-2 dagen voor)
───────────────────────────
□ Campaign objective bevestigd
□ Budget gedefinieerd (daily/lifetime)
□ Target audience gecreëerd
□ Creative assets goedgekeurd
□ Landing page live en getest
□ Tracking verified (pixel events)
□ UTM parameters correct
□ Naming convention toegepast

SETUP CHECKLIST
───────────────
□ Campaign level:
  ├── Correcte objective
  ├── Budget type (CBO vs ABO)
  ├── A/B test setup indien nodig
  └── Campaign budget ingesteld

□ Ad Set level:
  ├── Audience targeting correct
  ├── Placements ingesteld
  ├── Optimization goal correct
  ├── Budget (als ABO)
  ├── Schedule ingesteld
  └── Bid strategy gekozen

□ Ad level:
  ├── Creative correct uploaded
  ├── Primary text ingevuld
  ├── Headline ingevuld
  ├── CTA button correct
  ├── URL correct + UTMs
  └── Preview op alle placements

LAUNCH DAY
──────────
□ Final review in preview mode
□ Publiceer campagne
□ Verify delivery start
□ Note launch time voor rapportage
□ Set calendar reminder voor 24h check

POST-LAUNCH (24-48 uur)
────────────────────────
□ Delivery check (impressies komen binnen)
□ Geen policy rejections
□ CTR in verwachte range
□ Conversie tracking werkt
□ Learning phase gestart
```
