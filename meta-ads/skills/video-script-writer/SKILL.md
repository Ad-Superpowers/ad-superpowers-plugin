# Video Script Writer

## Overview

Deze skill genereert video scripts geoptimaliseerd voor Meta Ads, met focus op de cruciale eerste 3 seconden, effectieve storytelling structuren en platform-specifieke best practices voor maximale engagement en conversie.

## Hook Framework (Eerste 3 Seconden)

### Top 10 Video Hooks

| # | Hook Type | Script Voorbeeld | Visueel |
|---|-----------|------------------|---------|
| 1 | **Pattern Interrupt** | "STOP! Scroll niet verder..." | Bold tekst, beweging |
| 2 | **Shocking Statement** | "Dit gaat je €1000 besparen" | Geld visual, surprise |
| 3 | **Question Hook** | "Maak jij ook deze fout?" | Direct in camera |
| 4 | **Result Hook** | "Zo ging ik van X naar Y" | Before/after |
| 5 | **Demo Hook** | [Geen tekst, product in actie] | Product close-up |
| 6 | **Controversy** | "Experts hebben het fout over..." | Strong statement |
| 7 | **Relatable Pain** | "POV: Je [herkenbare situatie]" | Situatie naspelen |
| 8 | **Social Proof** | "10.000 mensen kochten dit al" | Testimonial montage |
| 9 | **Curiosity** | "Wat als ik je vertelde dat..." | Mysterieuze setup |
| 10 | **Tutorial Tease** | "Hier is hoe je [gewenst resultaat]" | Stap 1 laten zien |

### Hook Formaat Template

```
HOOK STRUCTUUR (0-3 sec):
├── VISUEL: [Wat ziet de kijker?]
├── AUDIO: [Wat hoort de kijker?]
├── TEKST: [On-screen tekst/caption]
└── ACTIE: [Wat gebeurt er?]

VOORBEELD:
├── VISUEL: Close-up gezicht, verbaasde uitdrukking
├── AUDIO: "Ik ontdekte iets geks..."
├── TEKST: "🤯 GAME CHANGER"
└── ACTIE: Snelle zoom-in naar product
```

## Video Script Structuren

### Structuur 1: Problem-Solution (15-30 sec)

```
SCRIPT TEMPLATE:
================

[0-3 sec] HOOK
├── Visuel: [Probleem situatie]
├── Audio: "Ben je ook zo moe van [probleem]?"
└── Tekst: "[PROBLEEM] 😤"

[3-10 sec] AGITATE
├── Visuel: [Frustratie tonen]
├── Audio: "Je hebt alles geprobeerd maar niets werkt..."
└── Tekst: [Geen tekst of subtiele pain point]

[10-20 sec] SOLUTION
├── Visuel: [Product reveal/demo]
├── Audio: "Tot ik [Product] ontdekte. Kijk wat het doet..."
└── Tekst: "✅ [Voordeel 1]" "✅ [Voordeel 2]"

[20-30 sec] CTA
├── Visuel: [Product + logo/website]
├── Audio: "Link in bio / Klik hieronder"
└── Tekst: "[AANBIEDING] 🔥 Shop Now"
```

### Structuur 2: Testimonial/UGC (15-45 sec)

```
SCRIPT TEMPLATE:
================

[0-3 sec] RELATABLE HOOK
├── Visuel: Persoon direct in camera
├── Audio: "Oké dus ik moet jullie iets vertellen..."
└── Tekst: [Geen - authentiek houden]

[3-15 sec] BACKSTORY
├── Visuel: B-roll van dagelijks leven
├── Audio: "Ik had altijd last van [probleem]..."
└── Tekst: [Subtitles]

[15-35 sec] DISCOVERY + RESULTS
├── Visuel: Product unboxing/gebruik
├── Audio: "Toen ontdekte ik [Product] en OMG..."
└── Tekst: "Week 1... Week 4... Nu!"

[35-45 sec] RECOMMENDATION
├── Visuel: Terug naar camera, enthusiast
├── Audio: "Als je [probleem] hebt, je MOET dit proberen"
└── Tekst: "[KORTING CODE] 💕"
```

### Structuur 3: How-To/Tutorial (30-60 sec)

```
SCRIPT TEMPLATE:
================

[0-3 sec] RESULT HOOK
├── Visuel: Eindresultaat tonen
├── Audio: "Zo krijg je [gewenst resultaat] in [tijd]"
└── Tekst: "TUTORIAL 📚"

[3-10 sec] INTRO + CREDIBILITY
├── Visuel: Presenter of product setup
├── Audio: "Ik ben [naam] en ik ga je laten zien..."
└── Tekst: "[Expertise/credential]"

[10-40 sec] STAPPEN
├── Visuel: Stap-voor-stap demonstratie
├── Audio: "Stap 1... Stap 2... Stap 3..."
└── Tekst: "STAP 1 ✅" "STAP 2 ✅" "STAP 3 ✅"

[40-55 sec] RESULT + TIPS
├── Visuel: Voor/na of eindresultaat
├── Audio: "En kijk! Pro tip: [extra waarde]..."
└── Tekst: "💡 PRO TIP"

[55-60 sec] CTA
├── Visuel: Product + call to action
├── Audio: "Wil je hetzelfde resultaat? Link in bio"
└── Tekst: "[CTA] ➡️"
```

### Structuur 4: Listicle (15-30 sec)

```
SCRIPT TEMPLATE:
================

[0-3 sec] CURIOSITY HOOK
├── Visuel: Bold nummer of countdown
├── Audio: "3 dingen die [doelgroep] niet weet over..."
└── Tekst: "3️⃣ GEHEIMEN"

[3-10 sec] PUNT 1
├── Visuel: Illustratie/demo punt 1
├── Audio: "Nummer 1: [Punt met waarde]"
└── Tekst: "#1 [Kernwoord]"

[10-17 sec] PUNT 2
├── Visuel: Illustratie/demo punt 2
├── Audio: "Nummer 2: [Punt met waarde]"
└── Tekst: "#2 [Kernwoord]"

[17-25 sec] PUNT 3 (Beste voor laatste)
├── Visuel: Illustratie/demo punt 3
├── Audio: "En de belangrijkste: [Beste punt]"
└── Tekst: "#3 🔥 [Kernwoord]"

[25-30 sec] CTA
├── Visuel: Alle punten recap + product
├── Audio: "Meer tips? Volg voor meer / Link in bio"
└── Tekst: "[CTA]"
```

## Platform-Specifieke Scripts

### Instagram Reels Script

```
FORMAT: Vertical 9:16, 15-30 sec
STIJL: Native, casual, trending audio

[0-2 sec] HOOK
├── Trending audio/sound start
├── Text overlay: Hook statement
└── Quick movement/transition

[2-15 sec] CONTENT
├── Fast pacing, snelle cuts
├── Native Reels transitions
└── Subtitles verplicht (85% kijkt zonder geluid)

[15-30 sec] PAYOFF + CTA
├── Satisfying conclusie
├── "Follow voor meer" of product link
└── Relevante hashtags in caption
```

### Facebook Feed Script

```
FORMAT: Square 1:1 of 4:5, 15-60 sec
STIJL: Professioneler, meer context

[0-3 sec] HOOK
├── Tekst-first (veel kijken zonder geluid)
├── Probleem of vraag stellen
└── Visually engaging opening

[3-45 sec] CONTENT
├── Kan langer en dieper
├── Mix van talking head + B-roll
└── Ondertiteling essentieel

[45-60 sec] CTA
├── Duidelijke call to action
├── Urgentie indien relevant
└── Clickable link reference
```

### Stories Script

```
FORMAT: Vertical 9:16, 5-15 sec
STIJL: Urgent, snappy, swipe-up focused

[0-2 sec] INSTANT HOOK
├── Beweging of text sticker
├── Poll/vraag element optioneel
└── Geen tijd te verliezen

[2-12 sec] CORE MESSAGE
├── 1 key message per story
├── Native stickers/GIFs
└── Interactie elementen

[12-15 sec] SWIPE CTA
├── Duidelijke swipe instructie
├── Arrow/gesture animation
└── Link sticker prominent
```

## Script Variatie Generator

**Vraag eerst:**
1. Wat is het product/dienst?
2. Welk platform? (Reels, Stories, Feed)
3. Gewenste lengte?
4. Type video? (UGC, product demo, testimonial, educational)
5. Funnel fase? (TOFU/MOFU/BOFU)

**Lever dan:**

```
VIDEO SCRIPT - [PRODUCT]
========================

📱 Platform: [Platform]

🎯 Doel: [Awareness/Conversion/etc]
```

## Voorbeeld Interactie

**Gebruiker**: "Ik heb een video script nodig voor een skincare serum. Instagram Reels, 20-30 seconden, UGC style. Het serum helpt tegen acne en is 100% natuurlijk."

**Claude response**:

"Hier is een UGC-style video script voor je skincare serum:

---

**VIDEO SCRIPT - Skincare Serum (Acne)**

📱 Platform: Instagram Reels
⏱️ Lengte: 25 seconden
🎯 Stijl: Authentieke UGC testimonial

---

**[0-3 sec] HOOK**
├── 🎬 Visuel: Close-up gezicht, finger pointing naar (oude) acne spots
├── 🔊 Audio: \"Oké ik MOET dit met jullie delen...\"
├── 📝 Tekst: \"acne journey 🧴\"
└── 💡 Tip: Kijk direct in camera, enthusiast maar authentiek

**[3-10 sec] BEFORE/PROBLEM**
├── 🎬 Visuel: Oude foto's/video's met acne (of beschrijven)
├── 🔊 Audio: \"Ik had jaren last van acne. Ik had echt ALLES geprobeerd...\"
├── 📝 Tekst: \"niks werkte 😭\"
└── 💡 Tip: Snel door oude content scrollen op telefoon

**[10-18 sec] DISCOVERY + SOLUTION**
├── 🎬 Visuel: Product unboxing/serum aanbrengen
├── 🔊 Audio: \"Tot iemand mij dit serum aanraadde. 100% natuurlijk, geen troep. Na 2 weken zag ik AL verschil.\"
├── 📝 Tekst: \"✨ 100% natuurlijk\" → \"week 2 ✅\"
└── 💡 Tip: Toon textuur, druppelaar, aanbrengen op huid

**[18-25 sec] RESULT + CTA**
├── 🎬 Visuel: Huidige huid close-up, stralend
├── 🔊 Audio: \"Nu? Kijk dan. Als jij ook struggled met acne - link in mijn bio. Trust me.\"
├── 📝 Tekst: \"🔗 link in bio\"
└── 💡 Tip: Zelfverzekerde glimlach, swipe up gesture

---

**PRODUCTIEPS:**
- Film bij natuurlijk licht (raam)
- Casual kleding, geen make-up heavy
- Authentieke toon > gepolijst
- Voeg trending audio toe in edit

Wil je nog een alternatieve versie met een andere hook?"
