# Ad Copy Generator

## Overview

Deze skill genereert effectieve ad copy voor Meta Ads door gebruik te maken van bewezen copywriting frameworks, psychologische triggers en platform-specifieke richtlijnen voor maximale conversie.

## Meta Ads Copy Structuur

### Anatomie van een Meta Ad

```
┌─────────────────────────────────────────────────────────────────┐
│  META AD COPY COMPONENTEN                                       │
│                                                                 │
│  PRIMARY TEXT (125-150 karakters zichtbaar)                     │
│  └── Max 2000 karakters, eerste ~125 meest belangrijk          │
│                                                                 │
│  HEADLINE (25-40 karakters ideaal)                              │
│  └── Max 255 karakters, kort & punchy werkt best               │
│                                                                 │
│  DESCRIPTION (optioneel, 20-30 karakters)                       │
│  └── Max 30 karakters, ondersteunend aan headline              │
│                                                                 │
│  CTA BUTTON (vast menu)                                         │
│  └── Shop Now, Learn More, Sign Up, etc.                       │
└─────────────────────────────────────────────────────────────────┘
```

## Hook Formules (Eerste Zin)

### De 3-Seconde Regel

```
Je hebt ~3 seconden om aandacht te pakken.
De eerste zin bepaalt of mensen stoppen met scrollen.
```

| # | Framework | Voorbeeld |
|---|-----------|-----------|
| 1 | **Probleem + Oplossing** | "Moe van [probleem]? Dit is de oplossing." |
| 2 | **Shocking Statistiek** | "80% van [doelgroep] maakt deze fout..." |
| 3 | **Vraag Stellen** | "Wist je dat [verrassend feit]?" |
| 4 | **Directe Aanspraak** | "Dit is voor jou als je [situatie]..." |
| 5 | **Social Proof** | "50.000+ mensen gingen je voor..." |
| 6 | **Urgentie** | "Alleen vandaag: [aanbieding]" |
| 7 | **Curiosity Gap** | "Het geheim achter [gewenst resultaat]..." |

| 9 | **Contrarian** | "Vergeet alles wat je wist over [topic]" |
| 10 | **Specifiek Resultaat** | "[Exact resultaat] in [exacte tijd]" |

## Copywriting Frameworks

### PAS Framework (Problem-Agitate-Solve)

```
STRUCTUUR:
P - Problem: Benoem het probleem
A - Agitate: Maak het probleem groter/pijnlijker
S - Solve: Presenteer jouw oplossing

VOORBEELD:
"Krijg je ook steeds hoofdpijn van Excel spreadsheets? [PROBLEEM]
Het kost je uren per week en je maakt constant fouten. [AGITATE]
Met [Product] automatiseer je alles in 5 minuten. [SOLVE]"
```

### AIDA Framework (Attention-Interest-Desire-Action)

```
STRUCTUUR:
A - Attention: Pak aandacht met hook
I - Interest: Wek interesse met voordelen
D - Desire: Creëer verlangen met resultaten
A - Action: Roep op tot actie

VOORBEELD:
"Stop met geld verspillen aan ineffectieve ads. [ATTENTION]
Onze AI analyseert 100+ datapunten per campagne. [INTEREST]
Klanten zien gemiddeld 3x hogere ROAS. [DESIRE]
Start je gratis trial →  [ACTION]"
```

### BAB Framework (Before-After-Bridge)

```
STRUCTUUR:
B - Before: Huidige situatie (pijn)
A - After: Gewenste situatie (plezier)
B - Bridge: Jouw product als brug

VOORBEELD:
"Je werkt 60 uur per week en ziet je gezin nauwelijks. [BEFORE]
Stel je voor: zelfde omzet, maar 20 uur minder werken. [AFTER]
[Product] automatiseert je workflows zodat jij tijd overhoudt. [BRIDGE]"
```

### 4U Framework

```
STRUCTUUR:
- Useful: Is het nuttig?
- Urgent: Is er urgentie?
- Unique: Is het uniek?
- Ultra-specific: Is het specifiek?

VOORBEELD:
"Bespaar 10 uur per week [USEFUL] - alleen deze week 50% korting [URGENT].
De enige tool die AI + human review combineert [UNIQUE].
327 bedrijven verhoogden hun conversie met 47% [ULTRA-SPECIFIC]."
```

## Copy Per Funnel Fase

### TOFU Copy (Awareness)

```
DOEL: Educeren, interesse wekken, probleem bewust maken
TOON: Informatief, nieuwsgierig makend, laagdrempelig
CTA: Learn More, Watch Video, Read More

VOORBEELD:
"Wist je dat 73% van de marketeers moeite heeft met [probleem]?

In deze video leggen we uit waarom dit gebeurt en
wat je eraan kunt doen. Geen sales pitch, gewoon waarde.

👆 Bekijk de video"
```

### MOFU Copy (Consideration)

```
DOEL: Vertrouwen bouwen, voordelen tonen, differentiëren
TOON: Helpend, expert, social proof
CTA: Sign Up, Get Started, Download, Try Free

VOORBEELD:
"Al 10.000+ marketeers gebruiken [Product] om hun
Meta Ads te optimaliseren.

✅ Gemiddeld 47% lagere CPA
✅ 3x snellere rapportage
✅ Automatische optimalisaties

Probeer 14 dagen gratis - geen creditcard nodig."
```

### BOFU Copy (Conversion)

```
DOEL: Conversie pushen, urgentie creëren, bezwaren wegnemen
TOON: Direct, urgent, resultaatgericht
CTA: Shop Now, Buy Now, Get Offer, Claim Discount

VOORBEELD:
"Je keek al 3x naar [Product].

Dit is je teken:
🔥 25% korting - alleen vandaag
🚚 Gratis verzending
↩️ 30 dagen retourrecht

Meer dan 5.000 ⭐⭐⭐⭐⭐ reviews.
Bestel nu →"
```

## Psychologische Triggers

### Triggers om te gebruiken in copy:

```
SCARCITY (Schaarste)
├── "Nog maar 5 op voorraad"
├── "Laatste dag van de sale"
└── "Beperkte oplage"

SOCIAL PROOF (Sociale bewijskracht)
├── "10.000+ tevreden klanten"
├── "⭐⭐⭐⭐⭐ (4.9/5 - 2.847 reviews)"
└── "Zoals gezien in [bekende media]"

AUTHORITY (Autoriteit)
├── "Aanbevolen door experts"
├── "Award-winning"
└── "10+ jaar ervaring"

RECIPROCITY (Wederkerigheid)
├── "Gratis [waardevolle content]"
├── "Geen creditcard nodig"
└── "Vrijblijvend advies"

LOSS AVERSION (Verliesaversie)
├── "Mis dit niet"
├── "Voorkom dat je..."
└── "Stop met geld verliezen aan..."

CURIOSITY (Nieuwsgierigheid)
├── "Het geheim achter..."
├── "Wat niemand je vertelt over..."
└── "De verrassende reden waarom..."
```

## Platform-Specifieke Tips

### Facebook vs Instagram vs Audience Network

```
FACEBOOK:
├── Langere copy kan werken (test!)
├── Meer context & detail toegestaan
├── Desktop users lezen meer
└── Emojis werken, maar subtiel

INSTAGRAM:
├── Korter = beter (visual first)
├── Emoji-vriendelijk
├── Lifestyle & aspirational toon
└── Minder dan 125 karakters ideaal

STORIES/REELS:
├── Ultra kort (< 50 karakters)
├── Native, casual toon
├── Emoji als visuele elementen
└── Urgentie werkt goed
```

## Character Limits & Best Practices

### Optimale Lengtes

| Component | Minimum | Optimaal | Maximum |
|-----------|---------|----------|---------|
| Primary Text | 40 | 100-150 | 2000 |
| Headline | 15 | 25-40 | 255 |
| Description | 10 | 20-30 | 30 |

### Formatting Tips

```
✅ DO:
├── Gebruik line breaks voor leesbaarheid
├── Bullets/emoji voor scanbare lijsten
├── Één duidelijke CTA per ad
├── Test korte vs lange versies
└── Gebruik cijfers (specifiek > vaag)

❌ DON'T:
├── ALL CAPS (behalve 1 woord voor nadruk)
├── Te veel emojis (max 3-5)
├── Clickbait die niet waargemaakt wordt
├── Meerdere CTA's in één ad
└── Jargon dat doelgroep niet kent
```

## A/B Test Suggesties

### Wat te testen in copy:

```
TEST PRIORITEIT:
1. Hook/eerste zin (hoogste impact)
2. Headline (tweede hoogste)
3. CTA button keuze
4. Lengte (kort vs lang)
5. Toon (formeel vs casual)
6. Met/zonder emojis
7. Met/zonder social proof
8. Met/zonder prijs
```
