## Image Optimization

```
IMAGE REQUIREMENTS
==================

TECHNICAL SPECS:
+-- Minimum: 100x100 pixels (250x250 for apparel)
+-- Maximum: 64 megapixels
+-- Format: JPEG, PNG, GIF, BMP, TIFF
+-- Size: Max 16MB per image
+-- URL: HTTPS required

BEST PRACTICES:
---------------
[ ] White or neutral background
[ ] Product fills 75-90% of frame
[ ] No text, watermarks or logos on image
[ ] High resolution (min. 800x800 recommended)
[ ] Product well-lit, no shadows
[ ] Multiple angles via additional_image_link

COMMON DISAPPROVALS:
+-- Promotional text on image -> Remove text overlays
+-- Watermarks -> Use watermark-free versions
+-- Generic images -> Use actual product photos
+-- Placeholder images -> Replace with real photos
+-- Blurry images -> Use higher resolution
```
