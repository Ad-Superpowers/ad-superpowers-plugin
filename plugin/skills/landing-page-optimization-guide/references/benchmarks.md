## Category 7: Technical Performance (10 Points)

### Speed Benchmarks

| Load Time | Conversion Impact |
|-----------|-------------------|
| 1-2 seconds | Optimal |
| 2-3 seconds | 7% conversion loss |
| 3-5 seconds | 25% conversion loss |
| 5+ seconds | 40%+ conversion loss |

### Technical Checklist

| Element | Check | Points |
|---------|-------|--------|
| Page loads <3 seconds | ☐ | 4 pts |
| All tracking pixels firing | ☐ | 3 pts |
| SSL certificate (https) | ☐ | 2 pts |
| No broken elements | ☐ | 1 pt |
| **Total** | | **10 pts** |

### Required Tracking (by Platform)

| Platform | Required Tracking |
|----------|------------------|
| Meta Ads | Meta Pixel + Conversion API |
| Google Ads | Google Tag + Enhanced Conversions |
| LinkedIn | LinkedIn Insight Tag |
| TikTok | TikTok Pixel + Events API |
| GA4 | GA4 tag with conversion events |
