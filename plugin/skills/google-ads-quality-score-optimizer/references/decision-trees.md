## Ad Relevance Optimization

### Keyword-Ad Alignment

```
AD RELEVANCE IMPROVEMENT
═════════════════════════

PROBLEM: "Below Average" Ad Relevance

DIAGNOSIS:
──────────
□ Does your ad copy contain the exact keyword?
□ Is the ad group too broad (too many keywords)?
□ Does the intent of keyword and ad match?

SOLUTIONS:
──────────

1. KEYWORD IN HEADLINES
────────────────────────
BAD: Ad Group "Running Shoes"
├── Keyword: buy running shoes
└── Headline: "Athletic Footwear Online" ✗

GOOD:
├── Keyword: buy running shoes
└── Headline: "Buy Running Shoes - Shop Now" ✓

2. KEYWORD INSERTION (Dynamic)
───────────────────────────────
Headline: {KeyWord:Running Shoes} Online
├── Query "nike running shoes"
├── Shows: "Nike Running Shoes Online"
└── Fallback: "Running Shoes Online"

Watch out for:
├── Grammar issues
├── Competitor names
└── Strange long-tail queries

3. AD GROUP TIGHTENING
───────────────────────
BEFORE (Too broad):
Ad Group: Shoes
├── running shoes
├── hiking shoes
├── sneakers
└── work boots

AFTER (Focused):
Ad Group 1: Running Shoes
├── running shoes
├── running sneakers
└── buy running shoes

Ad Group 2: Hiking Shoes
├── hiking shoes
├── hiking boots
└── buy hiking shoes

4. INTENT MATCHING
───────────────────
Keyword: "best running shoes"
Intent: Comparison, research

BAD Ad Copy:
"Buy Running Shoes Now - Lowest Price"

GOOD Ad Copy:
"Top 10 Best Running Shoes 2025 - Compare & Choose"
```

## Quality Score Diagnosis

### QS Troubleshooting Workflow

```
QS DIAGNOSIS FRAMEWORK
═══════════════════════

STEP 1: IDENTIFY PROBLEM KEYWORDS
───────────────────────────────────
1. Keywords → Modify Columns
2. Add: Quality Score, Exp. CTR, Ad Rel., LP Exp.
3. Sort by QS (low to high)
4. Focus on keywords with spend and QS <5

STEP 2: ANALYZE COMPONENTS
────────────────────────────
Per keyword, check which component(s) are "Below Average":

┌────────────────────┬───────────────────────────────────────┐
│ Below Average      │ Primary Focus                        │
├────────────────────┼───────────────────────────────────────┤
│ Expected CTR       │ → Ad copy optimization                │
│                    │ → Extensions                          │
│                    │ → Ad testing                          │
├────────────────────┼───────────────────────────────────────┤
│ Ad Relevance       │ → Keyword-ad alignment                │
│                    │ → Ad group restructuring              │
│                    │ → Keyword insertion                   │
├────────────────────┼───────────────────────────────────────┤
│ Landing Page Exp.  │ → Page speed                          │
│                    │ → Content relevance                   │
│                    │ → Mobile experience                   │
├────────────────────┼───────────────────────────────────────┤
│ Multiple Below     │ → Major restructuring needed          │
│                    │ → Consider pausing keyword            │
└────────────────────┴───────────────────────────────────────┘

STEP 3: PRIORITIZE ACTIONS
────────────────────────────
Priority Matrix:

HIGH PRIORITY (Fix First):
├── High spend + Low QS
├── Core keywords with Below Average
└── Brand keywords with QS issues

MEDIUM PRIORITY:
├── Medium spend + Average components
├── Non-core keywords
└── Testing candidates

LOW PRIORITY:
├── Low spend keywords
├── Long-tail with few impressions
└── Already Average/Above Average
```

### QS Benchmarks per Industry

```
QUALITY SCORE BENCHMARKS
════════════════════════

AVERAGE QS PER INDUSTRY:
──────────────────────────
├── E-commerce (Retail): 6-7
├── B2B Services: 5-6
├── Legal: 4-5
├── Finance/Insurance: 4-6
├── Healthcare: 5-6
├── Real Estate: 5-6
├── Travel: 5-7
└── Education: 6-7

GOALS (Realistic):
───────────────────
├── Brand Keywords: QS 8-10
├── Core Non-brand: QS 6-8
├── Competitive Keywords: QS 5-7
├── Long-tail Keywords: QS 5-7
└── Account Average: QS 6+

NOTE:
├── QS 10 on all keywords = not realistic
├── Focus on keywords that matter (spend)
├── QS is relative (you vs competitors)
└── QS improvements take time (weeks)
```
